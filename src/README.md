# Source Code
Folder ini merupakan source code dari project ini, `main.cpp` merupakan file utama di mana `int main()` berada dan akan ngehandle banyak proses.

Folder `modules` berisi semua module seperti:
- Beverages
- Desserts
- Cooking Process
- Order History
- New Order
- Utility toolset

Project ini dikerjakan oleh beberapa orang, silakan liat [Commit History](https://github.com/noaione/opercooked/commits/master) untuk Info lebih lanjut.
